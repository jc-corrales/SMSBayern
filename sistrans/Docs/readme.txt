1. Tener instalado la versión más reciente de Eclipse (Oxygen en el momento en el que
se desarrolló esta aplicación.
2. Tener un servidor WildFly 10.x instalado.
3. Tener en eclipse JBoss instalado.
4. Configurar el servidor de WildFly.
5. Correr el programa en el servidor.
6. Ejecutar Postman, para los servicios rest.
7. Cargas las colecciones en la carpeta de colecciones.
8. Probar las colecciones.
